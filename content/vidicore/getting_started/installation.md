To install VidiCore:

{{< tabs >}}
  {{< tab "Linux" >}}
Install using `apt`:
```sh
$ apt get install vidispine-server
```
  {{</ tab >}}

  {{< tab "Windows" >}}
Use Chocolatey package manager to install:
```powershell
$ choco install VidiCore
```
  {{</ tab >}}
{{</ tabs >}}

Then follow the setup guide.
